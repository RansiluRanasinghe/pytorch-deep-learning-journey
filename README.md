## Overview

This repository documents a structured, end-to-end learning journey through modern deep learning using PyTorch. Every notebook and project here was built with a focus on **understanding the fundamentals**, not just running code — from tensor operations and autograd to deploying production-ready models.

The goal: build a rock-solid foundation in deep learning that translates directly to real-world applications.

---

## Objectives

- Master core deep learning concepts and neural network architectures
- Build and train models using PyTorch from scratch
- Apply transfer learning and fine-tuning on pre-trained models
- Work on real-world Computer Vision and NLP tasks
- Understand model optimization and deployment pipelines

---

## Topics Covered

| Area | Topics |
|---|---|
| **PyTorch Fundamentals** | Tensors, Autograd, `nn.Module`, training loops |
| **Neural Networks** | MLPs, activation functions, loss functions, optimizers |
| **Computer Vision** | CNNs, image classification, data augmentation |
| **NLP** | Text classification, word embeddings, sequence models |
| **Transfer Learning** | Fine-tuning ResNet, EfficientNet, BERT |
| **Advanced Architectures** | ResNet, Transformers, Siamese Networks |
| **Model Optimization** | Pruning, quantization, mixed-precision training |
| **Deployment** | ONNX export, MLflow tracking |

## Learning Approach

- **From scratch first** — implemented core components manually before using high-level APIs
- **Transfer learning** — adapted pre-trained models to new tasks and domains
- **Hyperparameter tuning** — systematically improved model performance through experimentation
- **Architecture comparison** — benchmarked different approaches to understand trade-offs
- **Practical focus** — emphasized building working systems, not just academic understanding
